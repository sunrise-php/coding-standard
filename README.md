## Installation

```bash
composer require sunrise/coding-standard
```

## How to use?

```xml
<?xml version="1.0"?>
<ruleset name="Sunrise Coding Standard">
    <rule ref="./vendor/sunrise/coding-standard/ruleset.xml"/>

    <file>src</file>
    <file>tests</file>
</ruleset>
```
